<h1 align="center">ZyntraChain — Web3 UI Library</h1>

An open-source Web3 library of vector-based UI components for blockchain and Web3 applications, designed to help developers and designers build consistent, user-friendly decentralized products for the community.

## 📦 What's Inside

UI Components and 650+ Icons for the most popular cryptocurrencies and blockchain projects, all clean, consistent, and ready to use in your Web3 projects.

## 🚀 Usage

Simply copy any PNG/SVG from the `SVG Version` folder and use it in your project:

```html
<img src="SVG Version/Bitcoin (BTC).svg" width="32" height="32" />
```
