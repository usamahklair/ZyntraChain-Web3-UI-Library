## 📦 What's Inside

100+ SVG vector logos for the most popular cryptocurrencies and blockchain projects, all clean, consistent, and ready to use in your Web3 projects.

## 🚀 Usage

Simply copy any SVG from the `Vector Logos` folder and use it in your project:

```html
<img src="Vector Logos/Bitcoin (BTC).svg" width="32" height="32" />
```
